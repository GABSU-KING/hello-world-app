# 재무제표 탐색기

기업의 재무제표를 검색하고 주요 지표를 시각화하는 웹 애플리케이션입니다.

## 🚀 기능

- 기업 재무제표 검색
- 주요 재무 지표 시각화
- 반응형 디자인
- 현대적인 UI/UX

## 🛠️ 기술 스택

- React 18
- CSS3 (애니메이션, 그라디언트)
- HTML5
- API 연동 (예정)

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 빌드
npm run build
```

## 🌐 배포

이 프로젝트는 Vercel을 통해 배포됩니다.

## 📱 반응형 지원

- 데스크톱, 태블릿, 모바일 모든 디바이스 지원
- 터치 친화적 인터페이스

## 🎨 디자인 특징

- 그라디언트 배경
- 글래스모피즘 효과
- 부드러운 애니메이션
- 호버 효과
